Functions

spawn (cmd, sn_rules, callback)
Spawn a program. See awful.rules.execute for more details

Parameters:

  • cmd The command.
  • sn_rules a property table, false to disable startup-notification.
  • callback A callback function (if the client support startup notifications)

Returns:

  1. The forked PID or an error message
  2. The startup notification UID, if the spawn was successful
with_shell (cmd)
Spawn a program using the shell.

Parameters:

  • cmd The command.
with_line_callback (cmd[, stdout_callback[, stderr_callback[, done_callback]]])
Spawn a program and asynchronously and capture its output line by line.

Parameters:

  • cmd string or table The command.
  • stdout_callback function

    Function that is called with each line of

              output on stdout, e.g. <code>stdout\_callback(line)</code>.
    (optional)
  • stderr_callback function

    Function that is called with each line of

              output on stderr, e.g. <code>stderr\_callback(line)</code>.
    (optional)
  • done_callback function

    Function to call when no more output is

              produced.
    (optional)

Returns:

    Integer the PID of the forked process.

Or

    string Error message.
read_lines (input_stream, line_callback[, done_callback[, close=false]])
Read lines from a Gio input stream

Parameters:

  • input_stream Gio.InputStream The input stream to read from.
  • line_callback function

    Function that is called with each line

         read, e.g. <code>line\_callback(line\_from\_stream)</code>.
  • done_callback function

    Function that is called when the

              operation finishes (e.g. due to end of file).
    (optional)
  • close boolean Should the stream be closed after end-of-file? (default false)
pread (cmd)
Read a program output and returns its output as a string.

Parameters:

  • cmd The command to run.

Returns:

    A string with the program output, or the error if one occured.
