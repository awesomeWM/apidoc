Module awful.wibox

Wibox module for awful.

This module allows you to easily create wibox and attach them to the edge of a screen.

Info:

  • Copyright: 2009 Julien Danjou
  • Release: master-ge2cd174
  • Author: Julien Danjou <julien@danjou.info>

Functions

awfulwibox.get_position (wb) Get a wibox position if it has been set, or return top.
awfulwibox.set_position (wb, position, screen) Put a wibox on a screen at this position.
awfulwibox.attach (wb, position, screen) Attach a wibox to a screen.
awfulwibox.align (wb, align, screen) Align a wibox.
awfulwibox.stretch (wb, screen) Stretch a wibox so it takes all screen width or height.
awfulwibox.new (arg) Create a new wibox and attach it to a screen edge.


Functions

awfulwibox.get_position (wb)
Get a wibox position if it has been set, or return top.
  • wb The wibox

Returns:

    The wibox position.
awfulwibox.set_position (wb, position, screen)
Put a wibox on a screen at this position.
  • wb The wibox to attach.
  • position The position: top, bottom left or right.
  • screen If the wibox it not attached to a screen, specified on which screen the position should be set.
awfulwibox.attach (wb, position, screen)
Attach a wibox to a screen. If a wibox is attached, it will be automatically be moved when other wiboxes will be attached.
  • wb The wibox to attach.
  • position The position of the wibox: top, bottom, left or right.
  • screen The screen to attach to
awfulwibox.align (wb, align, screen)
Align a wibox.
  • wb The wibox.
  • align The alignment: left, right or center.
  • screen If the wibox is not attached to any screen, you can specify the screen where to align.
awfulwibox.stretch (wb, screen)
Stretch a wibox so it takes all screen width or height.
  • wb The wibox.
  • screen The screen to stretch on, or the wibox screen.
awfulwibox.new (arg)
Create a new wibox and attach it to a screen edge.
  • arg A table with standard arguments to wibox() creator. You can add also position key with value top, bottom, left or right. You can also use width or height in % and set align to center, right or left. You can also set the screen key with a screen number to attach the wibox. If not specified, 1 is assumed.

Returns:

    The wibox created.
