Class wibox.widget.background

A container capable of changing the background color, foreground color widget shape.

Info:

  • Copyright: 2010 Uli Schlachter
  • Release: master-g30eafb1
  • Author: Uli Schlachter

Methods

wibox.widget.background:set_widget (widget) Set the widget that is drawn on top of the background
wibox.widget.background:set_bg (bg) Set the background to use.
wibox.widget.background:set_fg (fg) Set the foreground to use.
wibox.widget.background:set_shape (shape) Set the background shape.
wibox.widget.background:set_shape_border_width (width) When a shape is set, also draw a border.
wibox.widget.background:set_shape_border_color ([fg=self.foreground]) When a shape is set, also draw a border.
wibox.widget.background:set_shape_clip (value) When a shape is set, make sure nothing is drawn outside of it.
wibox.widget.background:set_bgimage (image) Set the background image to use If image is a function, it will be called with (context, cr, width, height) as arguments.


Methods

wibox.widget.background:set_widget (widget)
Set the widget that is drawn on top of the background
  • widget widget The widget to be disaplayed inside of the background area
wibox.widget.background:set_bg (bg)
Set the background to use.

  • bg A color string, pattern or gradient (see gears.color)

Usage:

    local text_widget = {
    text   = 'Hello world!',
    widget = wibox.widget.textbox
}

parent : setup {
    {
        text_widget,
        bg     = '#ff0000',
        widget = wibox.widget.background
    },
    {
        text_widget,
        bg     = '#00ff00',
        widget = wibox.widget.background
    },
    {
        text_widget,
        bg     = '#0000ff',
        widget = wibox.widget.background
    },
    spacing = 10,
    layout  = wibox.layout.fixed.vertical
}

wibox.widget.background:set_shape (shape)
Set the background shape.

Any other arguments will be passed to the shape function

  • shape A function taking a context, width and height as arguments

Usage:

    parent : setup {
    {
        -- Adding a shape without margin may result in cropped output
        {
            text   = 'Hello world!',
            widget = wibox.widget.textbox
        },
        shape              = gears.shape.hexagon,
        bg                 = beautiful.bg_normal,
        shape_border_color = beautiful.border_color,
        shape_border_width = beautiful.border_width,
        widget             = wibox.widget.background
    },
    {
        -- To solve this, use a margin
        {
            {
                text   = 'Hello world!',
                widget = wibox.widget.textbox
            },
            left   = 10,
            right  = 10,
            top    = 3,
            bottom = 3,
            widget = wibox.layout.margin
        },
        shape              = gears.shape.hexagon,
        bg                 = beautiful.bg_normal,
        shape_border_color = beautiful.border_color,
        shape_border_width = beautiful.border_width,
        widget             = wibox.widget.background
    },
    spacing = 10,
    layout  = wibox.layout.fixed.vertical
}
wibox.widget.background:set_shape_border_width (width)

wibox.widget.background:set_shape_border_color ([fg=self.foreground])
When a shape is set, also draw a border.

See wibox.widget.background.set_shape for an usage example.

  • fg The border color, pattern or gradient (default self.foreground)
wibox.widget.background:set_shape_clip (value)
When a shape is set, make sure nothing is drawn outside of it.

  • value boolean If the shape clip is enable

Usage:

    parent : setup {
    {
        -- Some content may be outside of the shape
        {
            text   = 'Hello\nworld!',
            widget = wibox.widget.textbox
        },
        shape              = gears.shape.circle,
        bg                 = beautiful.bg_normal,
        shape_border_color = beautiful.border_color,
        widget             = wibox.widget.background
    },
    {
        -- To solve this, clip the content
        {
            text   = 'Hello\nworld!',
            widget = wibox.widget.textbox
        },
        shape_clip         = true,
        shape              = gears.shape.circle,
        bg                 = beautiful.bg_normal,
        shape_border_color = beautiful.border_color,
        widget             = wibox.widget.background
    },
    spacing = 10,
    layout  = wibox.layout.fixed.vertical
}
wibox.widget.background:set_bgimage (image)
Set the background image to use If image is a function, it will be called with (context, cr, width, height) as arguments. Any other arguments passed to this method will be appended.
  • image A background image or a function
