wibox.widget.textbox:set_markup (text)
Set the text of the textbox (with Pango markup).

Parameters:

  • text string The text to set. This can contain pango markup (e.g. <b>bold</b>). You can use awful.util.escape to escape parts of it.

Parameters:

  • text The text to display. Pango markup is ignored and shown as-is.
