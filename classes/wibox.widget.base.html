wibox.widget.base:make_widget (proxy[, widget_name])
Create a new widget. All widgets have to be generated via this function so that the needed signals are added and mouse input handling is set up.

Parameters:

  • proxy

    If this is set, the returned widget will be a proxy for this

              widget. It will be equivalent to this widget.
  • widget_name string Name of the widget. If not set, it will be set automatically via gears.object.modulename. (optional)
