Methods

wibox.layout.fixed:layout (context, width, height)
Layout a fixed layout. Each widget gets just the space it asks for.

Parameters:

  • context The context in which we are drawn.
  • width The available width.
  • height The available height.

Remove a widget from the layout

Returns:

    boolean index If the operation is successful
wibox.layout.fixed:remove_widgets (widget)
Remove one or more widgets from the layout The last parameter can be a boolean, forcing a recursive seach of the widget(s) to remove.

Parameters:

Returns:

    boolean If the operation is successful

Replace the first instance of widget in the layout with widget2

Parameters:

  • widget2 The widget to replace widget with

Swap 2 widgets in a layout If widget1 is present multiple time, only the first instance is swapped

Parameters:

  • widget1 The first widget
  • widget2 The second widget
  • recursive boolean Digg in all compatible layouts to find the widget. (default false)

    boolean If the operation is successful

wibox.layout.fixed:reset ()
Reset a fixed layout. This removes all widgets from the layout.
wibox.layout.fixed:fill_space (val)
Set the layout’s fill_space property. If this property is true, the last widget will get all the space that is left. If this is false, the last widget won’t be handled specially and there can be space left unused.

Parameters:

  • val
wibox.layout.fixed:horizontal (...)
Returns a new horizontal fixed layout. Each widget will get as much space as it asks for and each widget will be drawn next to its neighboring widget. Widgets can be added via :add() or as arguments to this function.

Parameters:

  • ... widget Widgets that should be added to the layout.
wibox.layout.fixed:vertical (...)
Returns a new vertical fixed layout. Each widget will get as much space as it asks for and each widget will be drawn next to its neighboring widget. Widgets can be added via :add() or as arguments to this function.

Parameters:

  • ... widget Widgets that should be added to the layout.
